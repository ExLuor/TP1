Choix de perturbations (0 = Aucune; 1 = Type1; 2 = Type2; etc.) et leur probabilité.
ErrType0 = Possibilité qu'un bit soit altéré dans la transmission.
ErrType1 = Possibilité de perdre complètement une trame.
ErrType2 = Possibilité d'interchanger l'ordre de deux trames.
[ERREURS]
ErrType0	0.25
ErrType1	0
ErrType2	0
[/ERREURS]

4 paramètre (c'est plus que 4 ça..)
1. La taille du tampon T-C du Thread C
2. La taille du tampon T-LLC utilisé de chaque côté du Thread C simulant le support de transmission (Protocole6).
3. Le délais de temporisation (time-out).
4. Le fichier à copier.
5. L'emplacement de destination pour la copie du fichier.
6. Choix de l'utilisation du code Hamming --> détecteur ou correcteur.
7. Les erreurs.
